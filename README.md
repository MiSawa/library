# みさわの暴力ライブラリ

- `graph/`
  - `cowgame.cc`
    牛ゲー.
  - `graph.cc`
    dijkstra, spfa, warshall-floyd, minimum spanning tree, minimum arborescence.

- `utils/`
  - `allocator.cc`
    継承すると new/delete が勝手にプールから取るように変わるやつ.
  - `easy_io.cc`
    cin, cout の wrapper.
  - `range.cc`
    python ぽい range.
  - `reverse.cc`
    rbegin(), rend() を保持するようなやつ.
  - `unroll.cc`
    自動ループ展開
